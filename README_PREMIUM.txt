Personel Sistemi Premium UI

Yapılan düzenlemeler:
- Boztek yazıları kaldırıldı.
- Server ile aynı premium logo eklendi.
- Ana ekrandaki uzun menüler aşağı doğru yığılmayacak şekilde 2 sütunlu Hızlı İşlemler kartına toplandı.
- Maaş, avans, kalan maaş ve yıllık izin kartları 2 sütunlu premium kart görünüme alındı.
- Koyu premium panel görünümü güçlendirildi.
- Mevcut QR giriş/çıkış ve API bağlantı yapısı korundu.

Android Studio ile açıp Build > Rebuild Project yapın.
